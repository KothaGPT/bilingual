## English

### Overview

**bilingual** is a Python package providing production-ready tools for Bangla and English natural language processing. It focuses on:

- ЁЯМН **Bilingual Support**: Equal treatment for Bangla and English
- ЁЯС╢ **Child-Friendly Content**: Special focus on educational and age-appropriate material
- ЁЯЪА **Production Ready**: Easy installation, comprehensive docs, robust testing
- ЁЯФз **Flexible**: From tokenization to translation, generation to classification
- ЁЯУЪ **Well-Documented**: Full documentation in both English and Bangla

### Features

- **Text Normalization**: Unicode normalization, punctuation handling, script cleaning
- **Tokenization**: Shared SentencePiece tokenizer optimized for Bangla + English
- **Language Models**: Bilingual pretrained and fine-tuned models for generation
- **Translation**: Bangla тЖФ English translation assistance
- **Classification**: Readability scoring, age-level detection, safety filtering
- **Utilities**: Dataset tools, evaluation metrics, preprocessing pipelines

#### Basic Usage

```python
from bilingual import bilingual_api as bb

# Load tokenizer
tokenizer = bb.load_tokenizer("bilingual-tokenizer")

# Normalize text
text_bn = bb.normalize_text("ржЖржорж┐ рж╕рзНржХрзБрж▓рзЗ ржпрж╛ржЪрзНржЫрж┐ред", lang="bn")
text_en = bb.normalize_text("I am going to school.", lang="en")

# Generate text
prompt = "A short story about a brave rabbit / рж╕рж╛рж╣рж╕рзА ржЦрж░ржЧрзЛрж╢рзЗрж░ ржПржХржЯрж┐ ржЫрзЛржЯ ржЧрж▓рзНржк"
story = bb.generate(prompt, model_name="bilingual-small-lm", max_tokens=150)

# Translate
translation = bb.translate("ржЖржорж┐ ржмржЗ ржкржбрж╝рждрзЗ ржнрж╛рж▓рзЛржмрж╛рж╕рж┐ред", src="bn", tgt="en")
print(translation)  # "I love to read books."

# Check readability
level = bb.readability_check(text_bn, lang="bn")
print(f"Reading level: {level}")
```

#### CLI Usage
